# vsscan

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts -h
```

To compile to an executable file (All-in-One):

```bash
bun compile
# you will find the output in:
(Mac | Linux) > ./output/vscan
(win)         > ./output/vscan.exe
```

To execute:

```bash
(Mac | Linux) > ./output/vscan -h
(win)         > ./output/vscan.exe -h
```
